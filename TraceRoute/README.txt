Compile the program using

javac tsapp.java
java tsapp [-c or -s or -p] [options] [ports]

Steps to run the application.
1) Run the server first with its respective arguments.
2) Run the proxies one by one next with its respective arguments.
3) Run the client lastly with its respective arguments.

The output is displayed only in client.

Sample inputs and outputs are given in the RFCTsapp.pdf file.
